# Wemsur's Personal Homepage

This is a personal homepage project built with [Next.js](https://nextjs.org/), [Tailwind CSS](https://tailwindcss.com/), and [shadcn/ui](https://ui.shadcn.com/), featuring a custom "Tweakcn" inspired vibrant theme.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Customization

The theme is defined in `app/globals.css`. You can adjust the CSS variables there to change the color palette.

## Components

The UI components are located in `components/ui` and are built on top of Radix UI primitives.
